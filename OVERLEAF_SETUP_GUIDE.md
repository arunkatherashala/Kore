## 🎨 Customize (Optional)

### **Change Title**
Find this in the LaTeX:
```latex
\title{
    \textcolor{darkblue}{\Huge KORE} \\
    \Large The Multi-Platform High-Performance \\
    \Large File Compression Format
}
```

Edit to your preference, recompile! ✅

### **Change Colors**
```latex
% Find these lines at top:
\definecolor{darkblue}{rgb}{0.1, 0.45, 0.91}
\definecolor{darkgreen}{rgb}{0.2, 0.65, 0.32}
\definecolor{darkred}{rgb}{0.8, 0.0, 0.0}

% Adjust RGB values (0-1 range)
% Click recompile to see changes
```

### **Change Font Size**
```latex
% At very top:
\documentclass[12pt,a4paper]{report}

% Change 12pt to:
% 10pt (smaller)
% 11pt (normal)
% 13pt (larger)
% 14pt (extra large)
```

### **Add Your Name/Logo**
After `\documentclass` line, add:
```latex
\usepackage{graphicx}

% Later in document:
\includegraphics[width=2cm]{your-logo.png}
```

Then upload logo image to project.

## 🔧 Troubleshooting

### **PDF Won't Compile**

**Error:** Red error message on right

**Solution:**
1. Check LaTeX syntax (look for red underlines)
2. Click error message for help
3. Overleaf shows exact problem
4. Usually missing `\end{document}` or bracket

**Fix:** The provided LaTeX is tested and should work fine!

---

### **PDF Looks Wrong**

**Problem:** Formatting weird, colors missing

**Solution:**
1. Click "Recompile" again
2. Wait for full compilation
3. Clear cache: Menu → Settings → Turn off Draft mode
4. Recompile

---

### **Can't See PDF**

**Problem:** No PDF on right side

**Solution:**
1. Wait 30+ seconds (first compile is slow)
2. Click green "Recompile" button
3. Check error messages (red text on right)
4. If still stuck, refresh page (F5)
