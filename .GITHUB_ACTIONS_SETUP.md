# GitHub Actions - PyPI Trusted Publisher Setup

## Overview
This workflow automatically publishes Kore to PyPI whenever you create a release tag.

## Setup Instructions (One-Time Only)

### Step 1: Configure GitHub Actions Secret (If Needed)
GitHub Actions already has `GITHUB_TOKEN` for releases. PyPI will use OpenID Connect for trusted publishing.

### Step 2: Set Up PyPI Trusted Publisher
This is the key step - you configure PyPI to trust GitHub Actions:

1. Go to https://pypi.org/manage/account/publishing/
2. Click **"Add a new pending publisher"** (under GitHub tab if not already selected)
3. Fill in the form:
   - **PyPI Project Name:** `kore-fileformat`
   - **Owner:** `arunkatherashala`
   - **Repository name:** `Kore`
   - **Workflow name:** `.github/workflows/publish-pypi.yml`
   - **Environment name:** (leave blank or use `pypi`)
4. Click **"Add"**

PyPI will show a "Pending publisher" until the workflow first runs successfully.

### Step 3: Create a Release to Trigger Deployment
Once everything is configured:

```bash
# Make sure changes are committed
git add .
git commit -m "feat: Add GitHub Actions for automated PyPI publishing"
git push origin main

# Create a release tag
git tag -a v0.1.0 -m "Kore Format v0.1.0 - Complete 8-language ecosystem"
git push origin v0.1.0
```

That's it! The workflow will:
1. ✅ Automatically trigger on the tag push
2. ✅ Build Python packages
3. ✅ Upload to PyPI using trusted publisher (no credentials needed!)
4. ✅ Create GitHub Release automatically

## How It Works
- **No API tokens stored** - Uses OpenID Connect trusted publishing
- **Automatic on tags** - Any `v*` tag triggers the workflow
- **Manual trigger available** - Actions tab → "Publish to PyPI" → Run workflow
- **Safe** - Only runs on official releases, not on every push

## Troubleshooting
- If workflow fails, check https://github.com/arunkatherashala/Kore/actions
- Make sure PyPI trusted publisher is configured
- First run might stay "Pending" - that's normal, it converts to active after first successful publish
